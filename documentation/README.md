# OMEN Command Center for Linux - Documentation

Welcome to the official developer documentation for the OMEN Command Center for Linux. This project is designed to bring native, hardware-level control for HP OMEN and Victus laptops to Linux, prioritizing a zero-trust security architecture, modularity, and high performance.

## Table of Contents

1. [Architecture & Communication Flow](1_architecture.md) - Understand how the GUI, Daemon, and Kernel drivers interact.
2. [Code Structure & File Locations](2_code_structure.md) - Detailed breakdown of where specific logic and code blocks reside.
3. [Adding Support for New Devices](3_adding_new_devices.md) - Step-by-step guide to adding support for unrecognised OMEN or Victus motherboards.
4. [Kernel Driver Internals (`hp-wmi.c`)](4_hp_wmi_internals.md) - Deep dive into the kernel driver's line-by-line structure and specific feature locations.
